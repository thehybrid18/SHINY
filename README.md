# R SHINY!: A MASTERPIECE

## Overview of the R SHINY library
Dual purpose: 

1. Dashboard solutions that can be written programmatically in an accessible html format.

2. Advance BI and data science capabilities within an organization.

## Extra Functionality 
Endless options with this ingenious product. 

1. Shiny is very scalable to compute needs. Shinyio gives users and corporations access to servers and auth login functionality. Visit [SHINYio](https://www.shinyapps.io/) for more information.

2. Shareable and easy to use because of the html format. It takes advantage of the very scalable BOOTSTRAP html styles, which tend to look good on mobile or any device.

3. Developers can use this framework to make very dynamic web pages without spending too much time on the underlying html code. Shiny has a feature for almost every html widget or application that you can think of. 

4. Options to place code in certain areas to update each R session or more frequently when a new user accesses the application. It is also easy to update and publish within the RSTUDIO IDE.
